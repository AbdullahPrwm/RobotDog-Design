🤖 Robotic Dog - Mechanical Design

📖 Overview

This project is a **simple mechanical design** for a quadruped robot (robotic dog).  
It was created as part of a summer training program in **Mechanical Engineering**.

The design is based on the YouTube tutorial **\[1DAY\_1CAD] ROBOT DOG SPOT** and was implemented **without any modifications** to the original design.

The goal is to understand the basic mechanical principles that allow a robot to stand and walk.
